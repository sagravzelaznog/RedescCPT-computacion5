Semana 6, Sesión 23: Mapeo de Red 🗺️

Uso de Advanced IP Scanner para Inventario y Detección de Hosts

1. ¿Por qué Necesitamos Mapear la Red?

El **Mapeo de Red** es la práctica de descubrir, identificar y documentar todos los dispositivos conectados a una red en un momento dado. Es una habilidad esencial para el mantenimiento y la seguridad:

Inventario y Auditoría

Saber cuántos PCs, impresoras y servidores existen, y a quién pertenecen. Esto facilita la gestión de IPs.

Seguridad

Detectar dispositivos desconocidos o intrusos conectados a la LAN. Un dispositivo no autorizado es un riesgo de seguridad inmediato.

2. Advanced IP Scanner: El Explorador Gratuito

**Advanced IP Scanner** es una herramienta gratuita y muy rápida, ideal para técnicos de campo. Escanea un rango de direcciones IP y devuelve información clave sobre los dispositivos que responden.

Información que Proporciona

  • **Estado:** Si el host está activo o inactivo.
  • **Dirección MAC:** Para identificar al fabricante del dispositivo.
  • **Nombre de Host:** El nombre de la computadora en la red.
  • **Puertos:** Muestra si puertos comunes (RDP, FTP, HTTP) están abiertos.

Configuración de Rango

Debes definir el rango de IP a escanear. Si tu red es 192.168.1.0/24, el rango será de **192.168.1.1 a 192.168.1.254**.

3. Práctica: Escaneo de tu Red Local

Tarea Práctica (Software Gratuito):

  1. Descarga e instala Advanced IP Scanner (o Angry IP Scanner).
  2. Identifica el rango de tu red local (ej. 192.168.1.1 a 192.168.1.254).
  3. Ejecuta el escaneo completo.
  4. Identifica los dispositivos conocidos (tu PC, router, móvil) y busca cualquier dispositivo con un nombre desconocido.
