La Línea de Comandos (CLI) y su Importancia en Servidores de Red

1. ¿Por Qué Linux es Clave en Redes?

**Linux** (y sus distribuciones como Ubuntu o CentOS) es el sistema operativo más utilizado para servidores web, firewalls, routers personalizados y servicios en la nube. Conocer su **Línea de Comandos (CLI)** es indispensable para la gestión de infraestructura de red.

Estabilidad y Seguridad

Linux es altamente estable, rara vez requiere reinicios y es menos propenso a virus que Windows, haciéndolo ideal para servicios críticos que deben estar siempre activos.

Bajo Consumo de Recursos

Las versiones de servidor (sin interfaz gráfica) consumen muy poca RAM y CPU, permitiendo que el hardware se enfoque en la tarea de servidor (ej. servir archivos o enrutar tráfico).

2. Comandos Básicos de Navegación y Archivos

El primer paso es familiarizarse con la estructura de directorios y los comandos básicos de manipulación de archivos:

Comandos de Navegación

  • **`pwd`** (Print Working Directory): Muestra dónde estás.
  • **`ls`**: Lista el contenido del directorio actual.
  • **`cd [directorio]`**: Cambia de directorio (ej. `cd /var/www`).

Comandos de Archivos

  • **`mkdir [nombre]`**: Crea un nuevo directorio.
  • **`touch [archivo]`**: Crea un archivo vacío.
  • **`cp [origen] [destino]`**: Copia archivos.
  • **`mv [origen] [destino]`**: Mueve o renombra archivos.

3. Tarea Práctica: Montando la Terminal Linux

Tarea Práctica (Preparación):

  1. Si usas Windows 10/11, activa y configura **WSL (Windows Subsystem for Linux)**, instalando una distribución (ej. Ubuntu).
  2. Si usas un OS diferente, instala una máquina virtual con **VirtualBox** o **VMware** y un Linux ligero.
  3. Practica los comandos básicos: `pwd`, `ls`, `cd /`, `mkdir prueba`.
