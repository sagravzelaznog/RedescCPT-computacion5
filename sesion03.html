Semana 1, Sesión 3: Componentes de Hardware 🛠️

Routers, Switches y Tarjetas de Interfaz de Red (NIC)

1. El Router: El Director de Tráfico 🧭

El **Router** es el dispositivo fundamental que conecta diferentes redes entre sí. Su función principal es **enrutar (routing)** los paquetes de datos por la mejor ruta hacia su destino.

Función Principal

  • **Conecta Redes Distintas:** Por ejemplo, conecta tu **Red Local (LAN)** con **Internet (WAN)**.
  • **Toma Decisiones:** Utiliza las **Direcciones IP** para decidir a dónde enviar un paquete.
  • **Tabla de Enrutamiento:** Mantiene una lista de rutas conocidas para tomar decisiones eficientes.

Analogía del Cartero

Imagina que el Router es un cartero experto. Recibe una carta (paquete de datos) y, basándose en la dirección completa (Dirección IP), sabe exactamente qué camino tomar (la mejor ruta) para que llegue a otra ciudad (otra red).

2. El Switch: El Organizador Local 🔗

El **Switch** opera dentro de una única red local (LAN). Su trabajo es asegurar que los datos enviados desde un dispositivo lleguen únicamente al dispositivo correcto, y no a todos los demás.

Función Principal

  • **Interconexión Local:** Conecta múltiples dispositivos finales (PCs, impresoras, servidores) dentro del mismo segmento de red.
  • **Uso de MAC:** Utiliza las **Direcciones MAC** (direcciones físicas únicas) para identificar con precisión a dónde enviar el paquete de datos, sin saturar la red.

Switch vs. Hub (Importante)

Antiguamente se usaban los **Hubs**, que eran "tontos": enviaban el dato a *todos* los puertos. El **Switch** es "inteligente": aprende qué dispositivo está en qué puerto, enviando el dato solo al puerto de destino, mejorando drásticamente el rendimiento.

3. Tarjeta de Interfaz de Red (NIC) 🏷️

La **NIC** (Network Interface Card) es el componente de hardware dentro de tu computadora o servidor que permite la conexión física a la red, ya sea por cable (Ethernet) o de forma inalámbrica (Wi-Fi).

Dirección MAC: Cada NIC tiene una dirección única de 48 bits grabada por el fabricante, conocida como **Dirección MAC** (Media Access Control). Es la "huella digital" del dispositivo, usada principalmente por el Switch.
