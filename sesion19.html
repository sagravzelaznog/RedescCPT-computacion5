Semana 5, Sesión 19: Router Doméstico 🏠

Acceso a la Interfaz, Contraseñas y Configuración de IP/Gateway

1. Acceso a la Interfaz Web del Router (Gateway)

El router SOHO (Small Office/Home Office) es el punto central de tu red. Para configurarlo, debes acceder a su interfaz web a través del navegador.

Dirección de Acceso

La IP del router es tu **Puerta de Enlace (Gateway)**. Las más comunes son **192.168.1.1** o **192.168.0.1**.

Simplemente escribe esta IP en la barra de direcciones de tu navegador.

Credenciales de Acceso

Los routers vienen con credenciales por defecto (ej. admin/admin o admin/password).

¡ERROR DE NOVICIO! Lo primero es cambiar estas credenciales para evitar accesos no autorizados a tu red.

2. Configuración Esencial del Router

Una vez dentro, hay configuraciones básicas que todo técnico debe saber manipular:

Configuración de WAN (Internet)

Es la interfaz que se conecta al proveedor de servicios de Internet (ISP). Puede ser por IP Dinámica (DHCP), Estática o PPPoE (común en líneas DSL).

Configuración de LAN (Red Local)

Aquí defines la **Dirección IP del Router** (el Gateway) y la Máscara de Red para tu red local. Es donde se define el rango IP que usarán todos tus dispositivos.

Configuración de Wi-Fi

Se establece el nombre de la red (**SSID**) y el tipo de seguridad (preferiblemente WPA2/WPA3).

3. Práctica: Acceso y Documentación

Tarea Práctica (Real/Simulada):

  1. Determina la IP de tu Gateway usando `ipconfig`.
  2. Accede a tu router por el navegador.
  3. Ubica la sección de "Administración" o "Seguridad" y verifica que las credenciales de administrador hayan sido cambiadas.
  4. Documenta el rango de IP de tu LAN.
