Creación de un Mapa de Red Profesional con Esquema de Direccionamiento

1. El Diagrama Lógico: El Alma de la Red

El **Diagrama Lógico** es una representación abstracta que muestra la estructura de la red en términos de direcciones IP, subredes, dominios de broadcast y cómo se agrupan los dispositivos. Para el proyecto PyME, es esencial mostrar el esquema de direccionamiento completo.

Capa de Abstracción

No se preocupa por dónde está el cable o el color; solo le importa que la **Red 192.168.10.0 /24** se conecte a Internet a través del **Gateway 192.168.10.1**.

Herramientas Gratuitas

Utiliza **Draw.io / Diagrams.net** o las capacidades de diagrama de **Packet Tracer** para crear tu mapa. Asegúrate de usar los símbolos estándar (ej. nube para Internet, cilindro para Servidor).

2. Elementos Clave en el Diagrama Lógico del Proyecto

Tu diagrama debe comunicar la siguiente información vital del proyecto:

El Router (Gateway)

Muestra las IPs de sus dos interfaces: la **IP Privada** (Gateway LAN) y la **IP Pública** (WAN/Internet, usa una ficticia como 203.0.113.1 si es necesario).

Los Rangos

Etiqueta la subred principal (`192.168.10.0/24`). Indica dónde se encuentran las **IPs Estáticas** (.1, .2, .3) y el rango de **DHCP** (ej. .50 - .75).

Conectividad Lógica

Muestra que los PCs (clientes DHCP) se conectan al Switch, y este conecta al Gateway, el cual se conecta a Internet.

3. Tarea: Creación del Diagrama

Tarea Práctica (Documentación):

  1. Abre Draw.io o Packet Tracer y dibuja los componentes principales.
  2. Etiqueta claramente la IP del Router, el Servidor y la Impresora.
  3. Utiliza flechas para mostrar el flujo de tráfico de un PC a Internet.
