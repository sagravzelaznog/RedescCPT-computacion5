Firewalls, Contraseñas Fuertes y Gestión de Puertos

1. El Firewall: El Guardia de Seguridad

Un **Firewall** (Cortafuegos) es un sistema de seguridad que monitorea y controla el tráfico de red entrante y saliente, basándose en reglas de seguridad preestablecidas. Es la primera línea de defensa de cualquier red.

Tipos de Firewalls

  • **Basado en Host:** Software instalado en la PC (ej. Firewall de Windows Defender).
  • **Basado en Red:** Dispositivo de hardware dedicado que protege toda la red (Router o Firewall dedicado).

Cómo Funciona

Trabaja inspeccionando el tráfico basándose en la IP, el Puerto y el Protocolo. Una regla simple es: "Permitir el tráfico saliente en el puerto 80 (HTTP), pero bloquear el tráfico entrante en ese puerto."

2. Reglas de Seguridad para el Técnico

La seguridad de red comienza con buenas prácticas. Como técnico, debes promover y aplicar estos hábitos:

Contraseñas Fuertes y Únicas

Nunca usar contraseñas por defecto. Combinar mayúsculas, minúsculas, números y símbolos. Usar una contraseña diferente para el Router, el Wi-Fi y el servidor.

Actualización de Firmware y Software

Las vulnerabilidades son fallos de seguridad que se corrigen con actualizaciones. Un router con firmware desactualizado es una invitación a la intrusión.

Puertos Abiertos (NAT/Port Forwarding)

Abrir puertos en el router debe ser el último recurso. Solo se deben abrir los puertos **estrictamente necesarios** para servicios específicos.

3. Tarea Práctica: Gestión del Firewall de Windows

Tarea Práctica (Windows):

  1. Abre la configuración del **Firewall de Windows Defender** (o el firewall de tu sistema operativo).
  2. Navega a la sección de "Reglas de entrada" y "Reglas de salida".
  3. Identifica las reglas que permiten el tráfico a tu navegador web.
  4. (Opcional): Intenta crear una regla simple para bloquear temporalmente el tráfico de un programa específico (¡Recuerda deshabilitarla después!).
